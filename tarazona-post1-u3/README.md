## Descripción del proyecto
Este proyecto consiste en el desarrollo de una página web de perfil personal utilizando HTML5 y CSS3.  

Se aplican conceptos como:
- Estructura semántica en HTML
- Selectores CSS
- Box Model (box-sizing: border-box)
- Posicionamiento (fixed, relative, absolute)
- Estilos para formularios accesibles

La página incluye:
- Sección de perfil
- Sección de habilidades
- Formulario de contacto

---

## Instrucciones de ejecución

1. Descargar o clonar el repositorio  
2. Abrir la carpeta en Visual Studio Code  
3. Ejecutar el archivo `index.html` con la extensión Live Server
